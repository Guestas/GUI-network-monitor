## Requirements
- [Node.js](https://nodejs.org/en/download/) (which comes with [npm](http://npmjs.com))

```bash
# mac homebrew
brew install node
```

## Install Dependencies

```bash
# Install dependencies
npm install
```

## Run the App

```bash
# Run the app
npm start
```

## Run the builder App

```bash
# Run the builder for app
npm run dist

npm run build:win
npm run build:mac
npm run build:linux

```
